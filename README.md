# How to Launch
## docker-compose
```bash
$ make up

$ make down
```

## sqlx
```bash
$ sqlx migrate add -r <name>

$ sqlx migrate run

$ sqlx migrate revert
```

## tailwind.css
```bash
$ cd client
$ npx tailwindcss -o ./tailwind.css
```

## launch
```bash
$ sh dev.sh
```
